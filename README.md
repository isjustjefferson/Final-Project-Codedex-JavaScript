# Final Project: Codédex JavaScript

Este é o projeto final da trilha de JavaScript da plataforma **Codédex**. O objetivo deste desafio foi aplicar os fundamentos da linguagem para criar lógica programática e interatividade.

---

## 🇧🇷 Português

### Descrição
O projeto marca a conclusão da jornada de JavaScript, onde foram aplicados conceitos essenciais para o desenvolvimento de software moderno. O foco foi estruturar uma solução que utiliza lógica condicional, laços de repetição e manipulação de dados para resolver problemas propostos pela plataforma.

### Tecnologias Utilizadas
* **JavaScript (ES6+)**: Lógica de programação e manipulação de variáveis.
* **Node.js**: Ambiente de execução para os scripts.

### O que este projeto demonstra:
* **Fundamentos de Programação**: Uso de variáveis, tipos de dados e operadores.
* **Controle de Fluxo**: Implementação de estruturas condicionais (`if/else`) e de repetição (`loops`).
* **Pensamento Algorítmico**: Capacidade de traduzir requisitos em código funcional.
* **Base para Backend**: Domínio da sintaxe necessária para evoluir para frameworks como Node.js e Express.

### Description
